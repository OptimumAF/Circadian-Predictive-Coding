# Support

## Getting Help

- Usage and setup questions: open a GitHub Discussion
- Bug reports: open a GitHub Issue using the bug report template
- Security issues: see `SECURITY.md`

## Before Opening A Question

Please include:

- Python version and OS
- Command you ran
- Full error output
- Relevant config flags

## Scope

This repository is research-oriented. Maintainers prioritize:

- Reproducible bugs
- Benchmark regressions
- Circadian algorithm behavior changes

General ML theory tutoring and custom model consulting are out of scope for issue support.
